## 🌟 **What is VAPOR?**

VAPOR is a **revolutionary Steam Grid Artwork Manager** that transforms your entire Steam library into a visual masterpiece with one click. No more manually searching for artwork - VAPOR automatically finds and installs the highest quality artwork for ALL your games in minutes.

### **🎯 Core Features**

- **🚀 One-Click Auto-Enhancement** - Transform your entire Steam library automatically
- **⚡ Lightning Fast Processing** - 1000+ games in under 25 minutes (3.5x faster than manual)
- **🎮 Steam Deck Optimized** - Perfect 1200×800 touch-friendly interface
- **🎨 Complete Artwork Suite** - Grid images, hero banners, logos, and icons
- **🌐 Universal Compatibility** - Windows, Steam Deck, Linux, macOS, portable installs
- **🔥 Unlimited Artwork** - Browse ALL available options (no artificial limits)
- **💎 Professional Quality** - Enterprise-grade performance with 90% cache hit rate

## 🚀 **Quick Start Guide**

### **📦 Installation (30 seconds)**
1. **[📥 Download Latest Release](../../releases)** - Get `VAPOR_v2.0.1.exe` (no Python needed!)
2. **🔑 Get Free API Keys**:
   - [Steam Web API Key](https://steamcommunity.com/dev/apikey) (30 seconds)
   - [SteamGridDB API Key](https://www.steamgriddb.com/profile/preferences/api) (1 minute)
3. **▶️ Run VAPOR** → Create Profile → **Auto-Enhance All Games** ✨

### **⏱️ Processing Speed**
| Library Size | Processing Time | Games per Minute |
|--------------|----------------|------------------|
| 100 games    | ~3 minutes     | ~33 games/min    |
| 500 games    | ~12 minutes    | ~42 games/min    |
| 1000+ games  | ~25 minutes    | ~40 games/min    |

*Performance varies by internet speed and SteamGridDB response times*

## 💻 **System Requirements**

### **Minimum Specifications**
- **Operating System**: Windows 10, Linux (any distro), Steam Deck, macOS 10.14+
- **Memory**: 4GB RAM (2GB during processing)
- **Storage**: 100MB for application + cache space
- **Network**: Internet connection for artwork downloads
- **Dependencies**: Steam installed and configured

### **Recommended for Optimal Performance**
- **Memory**: 8GB+ RAM for large libraries (1000+ games)
- **Network**: Broadband connection for faster downloads
- **Storage**: 500MB+ free space for artwork cache and temporary files
